Тема

Временные ряды (Time Series Analysis)

Кратко

Модели временных рядов используются для анализа исторических данных и прогнозирования будущего, например, для прогнозов продаж, спроса, цен на акции и т.д. Среди методов временных рядов выделяются ARIMA, GARCH, VAR и другие.

Подробно

Временные ряды — это последовательность данных, измеренных через равные промежутки времени. В финансах они используются для анализа и прогнозирования таких показателей, как цены акций, курсы валют, объемы торгов, инфляция и другие экономические индикаторы.

Основные компоненты временного ряда

  1. Тренд (Trend)

    2. Долгосрочное направление данных (рост, падение или стагнация).

    Пример: Устойчивый рост стоимости акций компании на протяжении 5 лет.

  2. Сезонность (Seasonality)

    3. Повторяющиеся паттерны в фиксированные периоды (день, месяц, квартал).

    Пример: Рост продаж ритейлеров в декабре из-за праздников.

  3. Цикличность (Cyclicity)

    4. Колебания без фиксированной периодичности, связанные с экономическими циклами.

    Пример: Спад рынка недвижимости каждые 7–10 лет.

  4. Шум (Noise)

    5. Случайные колебания, не объясняемые трендом, сезонностью или цикличностью.

Методы анализа временных рядов в финансах

Разложение ряда на компоненты:

1. Декомпозиция временного ряда


Y(t) = Trend(t) + Seasonality(t) + Noise(t)

Инструменты: STL (Seasonal-Trend decomposition using LOESS), метод Хольта-Винтерса.

2. Модели ARIMA (Autoregressive Integrated Moving Average)

Используется для несезонных стационарных рядов.

  • AR (Autoregression):

    • Зависимость от предыдущих значений.

    Y(t) = c + φ 1 Y(t - 1) + φ 2 Y(t - 2) + ... + φ p Y(t - p) + ε t

    Y(t) — текущее значение ряда

    φ1,φ2,...,φp — коэффициенты авторегрессии

    p — порядок авторегрессии (сколько прошлых значений учитывается)

    εt — ошибка (белый шум)

    c — константа

  • MA (Moving Average):

    • Зависимость от прошлых ошибок.

    Y(t) = c + ε t + θ 1 ε t - 1 + θ 1 ε t - 2 + ... + θ q ε t - q

    θ1,θ2,...,θq — коэффициенты скользящего среднего

    εt - 1εt - 2,... — ошибки в предыдущие моменты времени

    q — порядок скользящего среднего (сколько прошлых ошибок учитывается)

  • I (Integrated):

    • Количество дифференцирований для стационарности.

    Y′(t) = Y(t) − Y(t - 1)

Параметры: ARIMA(p, d, q):

p — порядок авторегрессии (сколько прошлых значений учитывается)

d — степень дифференцирования

q — порядок скользящего среднего (сколько прошлых ошибок учитывается)

3. Модели GARCH (Generalized Autoregressive Conditional Heteroskedasticity)

Используется для прогнозирования волатильности (например, рисков на фондовом рынке).

σ t 2 = α 0 + α 1 ε t - 1 2 + β 1 σ t - 1 2

σ2t - 1 — условная дисперсия (волатильность) в момент времени t.

α — это один из основных параметров GARCH-модели, который определяет степень AUTOКОРРЕЛЯЦИИ модели.

β — это вклад лаговой волатильности.

Для рассчитывания параметров α и β обычно используется алгоритм МАКСИМ.

4. Экспоненциальное сглаживание

  • Простое экспоненциальное сглаживание (SES):

    • Подходит для данных без тренда и сезонности.

  • Метод Хольта-Винтерса:

    • Учитывает тренд и сезонность.

    Y(t) = (Level + Trend) × Seasonality

5. Машинное обучение

  • LSTM (Long Short-Term Memory):

    • Рекуррентные нейросети для долгосрочных зависимостей.

  • Prophet (Meta):

    • Модель с автоматическим учетом трендов и сезонности.

  • XGBoost/LightGBM:

    • Градиентный бустинг для многомерных временных рядов.

Этапы построения модели:

  1. Сбор данных

    2. Пример: Ежедневные котировки акций (Open, High, Low, Close, Volume).

  2. Визуализация

    3. Построение графиков, выявление трендов и сезонности.

  3. Проверка на стационарность

    4. Тест Дики-Фуллера (ADF): Если p-value < 0.05, ряд стационарен.

    Преобразования: Логарифмирование, дифференцирование.

  4. Выбор модели

    5. На основе ACF/PACF-графиков или автоматического подбора (AutoARIMA).

  5. Обучение модели

    6. Разделение данных на тренировочную и тестовую выборки.

  6. Оценка качества

    7. Метрики: MAE, RMSE, MAPE.

  7. Прогнозирование

    8. Предсказание будущих значений на несколько шагов вперед.

Пример: Прогнозирование цены акций с помощью ARIMA

  1. Загрузка данных:

    2. Исторические данные Apple AAPL за 5 лет.

  2. Визуализация:

    3. График цен закрытия с явным восходящим трендом.

  3. Стационарность:

    4. После первого дифференцирования d=1 ряд становится стационарным ADF p-value = 0.01.

  4. Подбор параметров:

    5. ACF/PACF указывают на p=1, q=1.

  5. Модель ARIMA(1,1,1):

    6. Обучение на 80% данных.

  6. Прогноз:

    7. Предсказание цены на следующие 30 дней.

  7. Оценка:

    8. RMSE = 2.5$ на тестовой выборке.

Проблемы и ограничения

  • Нестационарность:

    • Требует преобразований (дифференцирование, логарифмирование).

  • Шумы:

    • Сложность прогнозирования в условиях высокой волатильности.

  • Выбор модели:

    • Риск переобучения или недообучения.

  • Экзогенные факторы:

    • Непредсказуемые события (кризисы, политические изменения).

Преимущества и недостатки

Преимущества Недостатки
Учет временных зависимостей Сложность работы с нестационарными рядами
Прогнозирование на основе истории Чувствительность к выбросам и шумам
Широкий выбор моделей (ARIMA, GARCH) Требует большого объема данных

Заключение

Анализ временных рядов — ключевой инструмент в финансах для прогнозирования цен, управления рисками и стратегического планирования. Однако его эффективность зависит от качества данных, правильного выбора модели и учета внешних факторов. Современные методы (например, LSTM) позволяют работать с нелинейными зависимостями, но классические подходы (ARIMA, GARCH) остаются востребованными из-за интерпретируемости.
