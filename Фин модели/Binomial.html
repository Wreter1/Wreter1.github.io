Тема

Модели оценки опционов (Black-Scholes, Binomial Model)

Кратко

Black-Scholes-Merton — одна из самых известных моделей для оценки справедливой стоимости европейских опционов. Биномная модель применяется для более сложных опционов с различными условиями исполнения, включая американского типа.

Подробно

Модель Блэка-Шоулза (Black-Scholes Model)

    Формула для оценки опционов:
  • Call-опцион:
    • C = S 0 × N( d 1 ) − K × e -rT × N( d 2 )
  • Put-опцион:
    • P = K × e -rT × N( -d 2 ) − S 0 × N( -d 1 )

Переменные:

S0 — Текущая цена базового актива.

K — Страйк-цена опциона.

T — Время до экспирации (в годах).

r — Безрисковая процентная ставка (непрерывно начисляемая).

σ — Волатильность базового актива (стандартное отклонение доходности).

N(x) — Функция распределения стандартной нормальной случайной величины.

d1 и d2 —

d 1 = ln( S 0 /K) + (r + σ 2 /2)T σ T
d 2 = d 1 − σ T

Основные предположения:

  1. Цена актива следует геометрическому броуновскому движению с постоянной волатильностью σ и дрейфом μ.
  2. Отсутствие транзакционных издержек и налогов.
  3. Непрерывная торговля и возможность коротких продаж.
  4. Постоянная безрисковая ставка (\( r \)) на протяжении срока действия опциона.
  5. Отсутствие дивидендов (но модель можно модифицировать для их учета).
  6. Опцион европейский (исполнение только в дату экспирации).

Преимущества:

Аналитическое решение, быстрое вычисление.

Широко используется на практике для европейских опционов.

Недостатки:

Не подходит для американских опционов (досрочное исполнение).

Предположение о постоянной волатильности и нормальном распределении не всегда выполняется.

Биномиальная модель (Binomial Model)

Построение дерева:

  1. Делим время до экспирации на n дискретных шагов Δt = T/n.
  2. На каждом шаге цена актива может вырасти в u раз или упасть в d раз:
    3. u = e σ Δt , d = 1/u
  3. Вероятность роста в риск-нейтральном мире:
    4. p = e rΔt − d u − d

Оценка опциона:

  1. Конечные узлы (экспирация):

    2. Для call: max (ST - K,0)

    Для put: min (K - ST,0)

    S T = S 0 × u k × d n - k

    k — число подъемов цены.

  2. Рекурсивный расчет (обратный проход):

    3. На каждом узле стоимость опциона вычисляется как дисконтированная ожидаемая стоимость:

    V = e -rΔt × [p × V up + (1 - p) × V down ]

    Для американских опционов сравнивают расчетную стоимость с выплатой при досрочном исполнении.

Преимущества:

Гибкость: подходит для американских опционов, учета дивидендов, изменяющихся параметров.

Наглядность: визуализация возможных сценариев.

Недостатки:

Вычислительная сложность при большом числе шагов.

Требует выбора количества шагов n, что влияет на точность.

Связь с Black-Scholes:

При n → ∞ биномиальная модель сходится к результату Black-Scholes.

Сравнение моделей

Критерий Black-Scholes Binomial Model
Тип опционов Только европейские Европейские и американские
Гибкость Ограничена предположениями Высокая (учет дивидендов, досрочного исполнения)
Скорость расчета Быстро (аналитическая формула) Медленнее (зависит от числа шагов n)
Точность Точна при выполнении предположений Приближенная, но улучшается с ростом n

Когда какую использовать:

  • Black-Scholes:

    • Для быстрой оценки европейских опционов с постоянными параметрами.

  • Binomial Model:

    • Для американских опционов, сложных условий (например, переменные дивиденды) или при необходимости визуализации сценариев.

Ключевое понятие:

Обе модели используют risk-neutral valuation — оценка основа
